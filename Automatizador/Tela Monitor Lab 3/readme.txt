# Automação da Planilha EAD

    Este script foi desenvolvido para automatizar processos dentro da planilha EAD. Siga as instruções abaixo para garantir que o fluxo de trabalho seja executado corretamente.

## Avisos Importantes:

    1. **Monitor do nosso PC no Lab 3**: Este script deve ser executado diretamente no monitor do nosso PC no lab térreo.

    2. **Sequência de Execução**:
        - **Primeiro**: Clique com o botão direito no script, edite no editpad++ e mude os valores das variáveis                               diaInicial,diaFInal, User (MSE) e Senha (MSE
        - **Segundo**: Execute o script.
        - **Terceiro**: Após o login, clique no SGF já logado.
        - **Quarto**: Clique no Opera e acesse a planilha EAD. Selecione o primeiro nome do aluno que deseja acompanhar.

    3. **Barra Inferior do Windows**: A barra inferior do Windows deve seguir a seguinte ordem de ícones:
        - **Windows**, **Explorer**, **Arquivos**, **Edge**, **Chrome**, **Opera**, **SGF**.
        - *Não altere a ordem desses ícones* para garantir o bom funcionamento do script.

    4. **Ordem das Abas no Opera**: 
        As abas no Opera devem estar na seguinte ordem:
        - **WhatsApp**
        - **SistemaEscola**
        - **Planilha EADs** (atualmente, ou a partir de janeiro 2025)

---

## Funcionalidades das Teclas de Atalho:

    - **F3**: Copia o nome do aluno, abre o contrato e exibe a porcentagem.
    - **F4**: Volta para a planilha e prepara para digitar a porcentagem manualmente.
    - **Shift + F4**: Similar ao F4, mas para a planilha EAD de janeiro de 2025, pois possui uma coluna a mais.
    - **F5**: Chama o aluno no WhatsApp. Envie a mensagem manualmente e copie a mensagem para a próxima ação.
    - **F6**: Função pendente.
    - **F7**: Registra a presença do aluno.
    - **F8**: Retorna para a planilha.
    - **Shift + F8**: Retorna para a planilha de janeiro de 2025, considerando a coluna adicional.
    - **F9**: Verifica o aluno no SistemaEscola. A presença deve ser verificada manualmente.
    - **F10**: Retorna para a planilha para que a presença ou falta seja digitada manualmente.
    - **Shift + F10**: Retorna para a planilha de janeiro de 2025, considerando a coluna a mais.
    - **Shift + F3**: Abre o contrato do aluno copiado e chama no WhatsApp.

---

## Instruções Gerais:

    1. **Antes de começar**: Certifique-se de que a ordem da barra inferior do Windows e das abas do Opera estejam corretas. Isso garante que as ações do script sejam executadas sem erros.

    2. **Durante a execução**: Siga a sequência correta de abas e ações conforme descrito acima. Use as teclas de atalho conforme a necessidade para agilizar o processo de acompanhamento do aluno.
